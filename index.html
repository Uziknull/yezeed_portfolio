Real Time Heart Rate Monitoring Device ( CARDIO-NODE )

This project involved designing and building a portable heart rate monitoring device that tracks pulse in real-time and measure the electrical activities of the heart. The device uses optical (PPG) sensors to detect blood flow changes and electrode (ECG) sensor to monitor the cardiac activities of the heart then transmits data to a web based dashboard via WiFi. It features alert systems for abnormal heart rates and stores historical data for health tracking.

Technologies Used:
ESP32, Pulse Sensor, ECG sensor, OLED display, 3D Printing
Project Duration:
2 months
Category:
Embedded Systems, IoT, Healthcare Tech
